# Vision Mart AI
### Automated Product Classification System

Vision Mart AI is a high-performance image recognition application developed to categorize products using Deep Learning. Built with a focus on speed and accuracy, the system leverages TensorFlow for real-time inference and Streamlit for a seamless user experience.

---

## Technical Stack
* **Framework:** Streamlit
* **Deep Learning:** TensorFlow (Keras)
* **Image Processing:** Pillow (PIL)
* **Numerical Computing:** NumPy
* **Language:** Python 3.10

---

## Key Features
* **Real-time Inference:** Rapid classification of uploaded images using pre-trained models.
* **Optimized for CPU:** Deployment-ready configuration using `tensorflow-cpu` for efficient resource management.
* **Interactive Dashboard:** User-friendly interface for uploading and analyzing product data.
